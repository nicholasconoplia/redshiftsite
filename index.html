Why RedShift?

Because f.lux and standard filters weren't enough.

True Red Filter

Unlike standard night modes that only go orange, RedShift allows for a deep red spectrum that truly minimizes blue light exposure.

Multi-Monitor Support

Works across all your screens on both Windows and macOS. No monitor left behind in the blue light zone.

Free & Open Source

No subscriptions, no trackers. Just code that works. Built for the community, by a developer who needed it.
